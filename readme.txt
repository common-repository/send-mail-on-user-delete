== Description ==
This plugin is used to send email notification to those users whose account has been deleted by admin.

You can also send email notification to admin.

== Screenshots ==

1. Email template

<strong>Feature</strong>

* Send email to deleted user
* Send email to site admin
* Edit email subject and content from back-end
* Write dynamic keyword using our available tokens in email template.
Example: %sitename%, %username%, %useremail% 


== Frequently Asked Questions ==
= How to edit email content? =
Go to "Dashboard -> SMOUD"

= Is there any dynamic keyword for sending email to deleted usere? =
Yes, you can use %sitename%, %username%, %useremail% for sending mail to deleted user.

== Installation ==
1. Upload the folder "send-mail-on-user-delete" to "/wp-content/plugins/" path.
2. Activate the plugin through the "Plugins" menu in WordPress backend
3. After activating the plugin, a new menu will be added "SMOUD". You can change email content there.
4. Enjoy!
